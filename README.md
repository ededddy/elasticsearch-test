# Experimental ElasticSearch setup in JavaScript

## Available Scripts

- Create cases & laws indices

```shell
yarn run add_cases
# or
npm run add_cases
```

- Add cases

```shell
yarn run add_cases
# or
npm run add_cases
```

- Add laws

```shell
yarn run add_laws
# or
npm run add_laws
```
